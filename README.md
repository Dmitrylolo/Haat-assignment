Haas assingment project for React Native delivery app

## Getting Started

### Prerequisites

- Node.js
- Yarn
- Android Studio

### Installation

1. Clone the repository
2. Install dependencies
   ```sh
   yarn install
   ```
3. Run the app
   ```sh
   yarn start
   ```
